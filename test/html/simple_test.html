Sending simple messages to the server and getting feedback

In this first demo, we are sending a simple message to the server with the send command. The message consists in a command, simple_test whose payload is an object containing the text input content . We don't excpect a reply from the server with the send

Node side 

	  
	  

	    The server will answer to the simple_test command
	    with a specific function handler. Message handlers
	    functions are stored in groups (that can be viewed as
	    plugins). When no group name is given to the
	    function install_mod, function handlers
	    are attached to the default "namespace" 0 group.
	  

	  

	    Each function handler's this value is set to
	    the websocket class. Its first argument is the
	    message object containing the data sent from the client.
